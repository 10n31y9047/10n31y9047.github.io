MarkDown参考手册

Markdown 的本质是让我们回归内容本身。 当你明白这个道理的时候，你会明白为什么许多功能 Markdown 没有。

1. 强调

使用星号*或者下划线_创建斜体或粗体。

*斜体* -> 斜体
**粗体** -> 粗体

2. 段落

段落是连续的几行文本，段落之间有一行或多行空白行。

这是第一段

这是第二段

若要添加换行符，请在行尾添加一个反斜杠\或两个空格 。

用反斜杠\\
换行

->

用反斜杠\
换行

3. 标题

在一行的开头用一个井号#加一个空格 创建一个标题。井号#越多，标题的级别就越低。

# 一级标题
## 二级标题
### 三级标题
…

->

一级标题

二级标题

三级标题

…

标题里也可以包含斜体或粗体那样的格式。

# *斜体一级标题*

->

斜体一级标题

4. 引用块

在一行的开头使用大于号>来创建一个引用块，大于号>后面可以加一个空格 ，也可以不加。

>这是一个引用块

->

这是一个引用块

若要保持引用块连续，在引用块内的空行必须以大于号>开头。

>这是引用块的第一段
>
>这是第二段，上面的空行必须以大于号>开头

->

这是引用块的第一段

这是第二段，上面的空行必须以大于号>开头

引用块可以嵌套。

引用块可以嵌套

就像这样

5. 列表

无序列表可以使用星号*、加号+或连字符-作为列表标记，每个标记符后面要有一个空格 。

* 无序列表1
* 无序列表2
* 无序列表3

->

  • 无序列表1
  • 无序列表2
  • 无序列表3

有序列表使用数字后跟点.或右括号)。

1. 有序列表1
2. 有序列表2
3. 有序列表3

->

  1. 有序列表1
  2. 有序列表2
  3. 有序列表3

如果你不想使用有序列表，在点.之前要加上反斜杠\。

1. 列表
2. 列表
3\. 不是列表

->

  1. 列表
  2. 列表
    3. 不是列表

6. 超链接

超链接的文本用方括号[]括起来。

超链接可以位于一行文本之中，位于文本行中的超链接，其链接要用小括号()括起来。

点击这里访问[Github](https://github.com\). -> 点击这里访问Github.

超链接也可以作为引用参考放在文本的末尾，方括号[]之间不能有空格 。链接的定义可以放在空行之后的任何地方，但通常放在一个段落的底部。定义标识符可以由字母、数字、空格和标点符号组成，它们不区分大小写。

文本中的 [链接1][1]和[链接2][two]

[1]: http://www.bing.com/\
[two]: http://www.github.com/

->

文本中的 链接1和链接2

链接可以有一个名称，也可以没有。

[Github](https://www.github.com/ "Github") -> Github

相对路径也可以使用。

[相对路径](/index.html) -> 相对路径

在某些Markdown中，除非加上尖括号<>，否则路径可能不会成为链接。

<https://www.github.com/\> -> https://www.github.com/

7. 图片

图片和超链接很像，只是在方括号[]之前多了一个感叹号!，其他语法相同。需要注意的是，图片的方括号[]内可以留空。

![我的头像](https://i.loli.net/2020/08/18/GA5I1T79BPDufl3.jpg "我的头像")

->

![](https://i.loli.net/2020/08/18/GA5I1T79BPDufl3.jpg\)

->

8. 代码块

代码块内的每个字符与键入的字符完全相同。

要在一行文本内创建代码块，可以把用反引号`包括起来。

文本中的代码块`Debug.Log(“Hello World”)`

->

文本中的代码块Debug.Log("Hello World")

若要创建单独的代码块，则要在代码块的上下两行分别输入三个反引号`，或者分别缩进四个空格 。若使用四个空格，则代码块将持续直到到达未缩进的行。

```
using System;
class Program
{
 static void Main(string[] args)
 {
  Console.WriteLine(“Hello World!”);
 }
}
```

->

using System;
class Program
{
   static void Main(string[] args)
   {
       Console.WriteLine("Hello World!");
   }
}

9. 分割线

使用三个连字符-或星号*创建分割线。

---

->

10. 嵌套列表

要将一个列表嵌套到另一个列表中，请将子列表中的每个项缩进四个空格。您还可以嵌套其他元素，如段落、引用块或代码块。

* 列表
 1. 子列表
 2. 子列表
* 列表
 - 子列表
 - 子列表
* 列表

->

  • 列表
    1. 子列表
    2. 子列表
  • 列表
    • 子列表
    • 子列表
  • 列表

11. 支持的HTML元素

不在 Markdown 涵盖范围之内的标签，都可以直接在文档里面用 HTML 撰写。目前支持的 HTML 元素有：<kbd> <b> <i> <em> <sup> <sub> <br>等。

使用 <kbd>Ctrl</kbd>+<kbd>Alt</kbd>+<kbd>Del</kbd> 重启电脑

->

使用 Ctrl+Alt+Del 重启电脑
